# ICA 11 - Flight Data Visualization

## Overview
This project analyzes airline passenger data and visualizes trends over time using line plots.

## Objective
To generate line plots for each month showing passenger counts across years.

## Files Included
- **Lutalo_Rodger_ICA 11.R** – R script used for analysis and plotting  
- **flights.jpg** – Final visualization output  

## Methods
- Data imported using `read.csv()`
- Visualization created using `ggplot2`
- Lines represent monthly trends over time

## Tools
- R
- ggplot2
- dplyr

## Output
The plot shows how passenger numbers change over time for each month, highlighting seasonal patterns and overall growth.
